# Lista de Exercícios - Prática de Funções em JavaScript

Esta pasta contém **7 exercícios práticos** organizados em três níveis distintos para aprendizado e prática de funções em JavaScript.

## Estrutura dos Exercícios

### **Nível Básico** (3 Exercícios)

-   **Exercício 1**: Saudação Simples - Declaração tradicional de função sem parâmetros
-   **Exercício 2**: Soma de Dois Números - Função com parâmetros e retorno de valor
-   **Exercício 3**: Verificar Par ou Ímpar - Uso de condicionais dentro de função

### **Nível Intermediário** (2 Exercícios)

-   **Exercício 4**: Inverter String - Manipulação de strings com métodos split(), reverse() e join()
-   **Exercício 5**: Contagem de Vogais - Iteração sobre strings e lógica de contagem

### **Nível Avançado** (2 Exercícios)

-   **Exercício 6**: Expressão de Função - Cálculo simples usando expressão de função (const nomeDaFuncao = function(...))
-   **Exercício 7**: Verificar se é Palíndromo - Manipulação de strings e lógica de comparação

## Como Executar

```bash
node 01.js  # Executa apenas o exercício 01
node 07.js  # Executa apenas o exercício 07
node 03.js  # Executa apenas o exercício 03
```

## Conceitos Abordados

-   [x] Declaração tradicional (`function nomeDaFuncao()`)
-   [x] Expressão de função (`const nomeDaFuncao = function()`)
-   [x] Funções com parâmetros
-   [x] Funções com retorno de valores
-   [x] Funções sem parâmetros

### Controle de Fluxo

-   [x] Estruturas condicionais (`if/else`)

-   [x] `reverse()` - Inverter array
-   [x] `join()` - Converter array em string
-   [x] `toLowerCase()` - Converter para minúsculas
-   [x] `replace()` - Substituir caracteres
-   [x] Acesso a caracteres por índice
-   [x] Iteração sobre strings

### Operações Matemáticas

-   [x] Operadores aritméticos (`+`, `*`)
-   [x] Cálculo de área de retângulo

### Algoritmos Clássicos

-   [x] Verificação de palíndromos
-   [x] Contagem de vogais
-   [x] Inversão de strings

### Formatação e Saída

-   [x] `console.log()` - Saída de dados
-   [x] Template literals (strings com `${}`)
-   [x] Formatação de mensagens
